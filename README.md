<h1 align="center">客户端性能测试平台</h1>
<ul>
    <li>支持android/ios平台上应用的性能数据测试</li>
    <li>支持指标包含:cpu,memory,fps,gpu,温度,电量</li>
    <li>fps指标中包含：jank,big_jank,增量耗时,等指标</li>
    <li>数据结果准确，与perfdog一致</li>
    <li>实时记录设备画面</li>
    <li>支持局域网内使用</li>
</ul>
